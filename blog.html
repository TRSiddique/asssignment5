What is the DOM in web development?

The DOM is a tree-like structure representing a webpage's HTML, allowing developers to manipulate the content, structure, and style using JavaScript.

How can I select an element by its id in the DOM?

You can select an element by its id using document.getElementById('yourId').

How do I add a new element to the DOM?

Use document.createElement('tag') to create a new element and appendChild() to add it to the DOM.

How do I change the style of an element using the DOM?

You can modify an element's style using the element.style property, for example: element.style.color = 'blue';.
